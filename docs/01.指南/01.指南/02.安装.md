## SpringBoot

### 🍊Maven
在项目的pom.xml的dependencies中加入以下内容:
```xml
<dependency>
    <groupId>icu.xwang</groupId>
    <artifactId>Xw-api-sdk</artifactId>
    <version>1.0.0</version>
</dependency>
```
### 🍐Gradle
```
implementation group: 'icu.xwang', name: 'Xw-api-sdk', version: '1.0.0'
```
